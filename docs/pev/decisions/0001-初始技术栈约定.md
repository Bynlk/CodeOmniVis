# 0001-初始技术栈约定

- **日期**：2026-07-07
- **状态**：已采纳
- **相关 feature**：全部

## 背景 / 触发原因
PEV 首次初始化，需登记本项目完整技术栈铁律。本期为 CodeOmniVis 前端重构，依据 docs/pev/managed/requirements/frontend-redesign-brief_v1.md。

## 决策
### 后端（项目硬约束，本期不改动）
- 后端框架必须含 **TSRPC**。
- 数据库必须含 **MongoDB**。
- 本期前端重构不触碰后端代码，仅消费既有接口；后端硬约束登记在册，未来后端开发须遵守。

### 前端（本期直接约束）
- 语言/框架：TypeScript + React 18（不可替换）。
- 图渲染：Cytoscape 3 + dagre（不可替换）。
- 样式：Tailwind 3（禁止外部 CDN）。
- 实时：原生 WebSocket。
- 构建：Vite 5 / tsup。
- i18n：i18next + react-i18next。
- 服务器状态：@tanstack/react-query。

### 数据契约
- OmniGraph / 17 NodeType / 15 EdgeType / 12 API 端点 100% 兼容，禁止破坏。

## 影响范围
- 约束全部 feature 的实现方式。
- 无返工。
