# 项目目标

**核心目标**：GitHub 影响力优先。

**优先级排序**：
1. 视觉效果 — demo 必须震撼
2. demo 体验 — 一键出图，60 秒内
3. 功能完整性 — 能用就行，不求完美
4. 解析深度 — 可以有误报，不能崩溃

**核心卖点**：`npx codeomnivis serve` → 60 秒内看到整个 TypeScript 全栈项目的架构图。

**目标用户**：
- 主要：加入新项目需要快速上手的全栈工程师
- 次要：AI 编码助手（通过 MCP 接口）、Tech Lead、开源贡献者

**Why:** 用户是 solo 开发者，目标是通过这个项目积累 GitHub star 和影响力，服务求职。

**How to apply:** 每个功能实现时，先问"这个对 demo 效果有贡献吗？"，优先做视觉相关的功能。
